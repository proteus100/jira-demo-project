---

## Step 2: Create Swimlanes (3 minutes)

### Create 3 Horizontal Rectangles (Lanes)

1. **Click the rectangle tool** (or press R)
2. Draw 3 wide rectangles stacked vertically
3. **Click the paint bucket** to fill with color:
   - Top lane (Developer): Light blue
   - Middle lane (GitHub Actions): Light green
   - Bottom lane (Jira): Light yellow

4. **Add text labels** (press T for text):
   - Click on each lane
   - Type: "DEVELOPER", "GITHUB ACTIONS", "JIRA SYSTEM"
   - Make text bold and larger

---

## Step 3: Add Process Boxes (8 minutes)

### Create Process Step Boxes

1. **Use rectangle tool** (R)
2. Draw small white rectangles in each lane:

**Developer Lane (3 boxes left to right):**
- "Create branch\nSCRUM-5"
- "Write code\n& commit"
- "Open Pull\nRequest"

**GitHub Actions Lane (3 boxes):**
- "Detect\nPR event"
- "Extract\nJira key"
- "Call Jira\nAPI"

**Jira Lane (3 boxes):**
- "Validate\nkey"
- "Transition\nto Review"
- "Add\ncomment"

3. **Style the boxes:**
   - Fill: White
   - Border: Dark/black
   - Round corners (use corner radius slider)

## Excalidraw Tips

**Keyboard Shortcuts:**
- **R** = Rectangle
- **A** = Arrow
- **T** = Text
- **V** = Selection tool
- **Ctrl/Cmd + D** = Duplicate
- **Ctrl/Cmd + Z** = Undo

**Styling:**
- Use the **property panel on the left** to change colors, stroke, etc.
- **Double-click any element** to edit text
- **Drag to select multiple elements** then style all at once

**Hand-Drawn Style:**
- This is a FEATURE not a bug!
- Low-fidelity wireframes are industry standard
- Shows you're focused on concepts, not pixel-perfect design
